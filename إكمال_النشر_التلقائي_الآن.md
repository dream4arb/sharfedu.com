# إكمال النشر التلقائي — خطوات الآن

بعد تنفيذ هذه الخطوات **مرة واحدة**، أي **push** من المحلي سيحدّث الموقع تلقائياً خلال حوالي دقيقة.

---

## التحقق أولاً (قد يكون جاهزاً)

- **`.deploy-token`** في **public_html**: تمت استعادته في المرحلة 1. إن كنت قد أنشأته سابقاً فهو موجود.
- **deploy-webhook.php** في **public_html**: موجود لأنه نُسخ من `deploy_public_html` في المرحلة 1.

---

## الخطوة أ — إضافة الـ Cron على السيرفر (SSH)

1. افتح SSH وادخل السيرفر.
2. نفّذ:
   ```bash
   crontab -e
   ```
3. إذا طلب اختيار محرر، اختر رقم **nano** (مثلاً `1`) واضغط Enter.
4. في نهاية الملف أضف **سطراً واحداً** (انسخه كاملاً):

   ```
   * * * * * cd /home/894422.cloudwaysapps.com/cmkdrtgqcv/git_repo && [ -f ../public_html/deploy-requested.flag ] && git fetch origin && git reset --hard origin/main && bash scripts/deploy-on-server.sh && rm -f ../public_html/deploy-requested.flag
   ```

5. احفظ واخرج:
   - في **nano**: `Ctrl+O` ثم Enter، ثم `Ctrl+X`.

---

## الخطوة ب — إضافة Secretين في GitHub

1. افتح المستودع على GitHub: **https://github.com/tariqalmarifa/sharfedu.com**
2. **Settings** → **Secrets and variables** → **Actions**
3. **New repository secret** وأضف:

   | الاسم | القيمة |
   |------|--------|
   | `DEPLOY_WEBHOOK_URL` | `https://sharfedu.com/deploy-webhook.php` |
   | `DEPLOY_WEBHOOK_TOKEN` | **نفس الرمز** الموجود داخل ملف `.deploy-token` على السيرفر |

لرؤية الرمز على السيرفر (SSH):

```bash
cat /home/894422.cloudwaysapps.com/cmkdrtgqcv/public_html/.deploy-token
```

انسخ المحتوى كما هو (سطر واحد) والصقه في قيمة الـ Secret `DEPLOY_WEBHOOK_TOKEN`.

---

## التحقق أن كل شيء يعمل

1. من المحلي: أي تعديل بسيط → **npm run build** → **git add** الملفات المهمة (مثل `src` و `deploy_public_html`) → **git commit** → **git push origin main**.
2. في GitHub: **Actions** — يفترض أن يظهر workflow **Deploy to Cloudways** بلون أخضر.
3. خلال **حوالي دقيقة**: افتح **https://sharfedu.com** وتحقق من ظهور التعديل (وإن أردت: **https://sharfedu.com/last-deploy.txt** لرؤية وقت آخر نشر).

---

## ملخص

| الخطوة | أين | ماذا |
|--------|-----|------|
| أ | السيرفر (SSH) | `crontab -e` ثم إضافة السطر الذي يشغّل النشر كل دقيقة عند وجود العلم |
| ب | GitHub → Settings → Secrets | إضافة `DEPLOY_WEBHOOK_URL` و `DEPLOY_WEBHOOK_TOKEN` |

بعدها: **تعديل محلي → build → push** = تحديث الموقع تلقائياً خلال حوالي دقيقة.
